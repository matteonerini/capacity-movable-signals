# Capacity of Two-User Wireless Systems Aided by Movable Signals

This repository contains code to reproduce the paper:

> M. Nerini, B. Clerckx, "[Capacity of two-user wireless systems aided by movable signals](https://ieeexplore.ieee.org/document/11615290)," IEEE Wireless Commun. Lett., 2026.

## Code

The script `main_Fign.m` reproduces Fig. *n*, for *n* = 2, 3, 4.
